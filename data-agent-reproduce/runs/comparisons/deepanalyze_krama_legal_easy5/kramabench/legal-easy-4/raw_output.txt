The task is to count the total number of fraud reports made by the FTC over the web between 2022 and 2024.

From the provided data file `2024_CSN_Data_Contributors.csv`, I need to:
1. Identify rows where the Data Contributor contains "FTC - Web Reports (Fraud & Other)".
2. Extract the number of reports for each year (2022, 2023, 2024).
3. Sum these numbers to get the total fraud reports over the web during the specified period.

Looking at the data:
- For 2022: "FTC - Web Reports (Fraud & Other)" has 693,789 reports.
- For 2023: "FTC - Web Reports (Fraud & Other)" has 702,598 reports.
- For 2024: "FTC - Web Reports (Fraud & Other)" has 715,248 reports.

I will write a Python script to parse the CSV file and extract the required values.
</Analyze>

<Code>
```python
import csv

# Initialize total fraud reports
total_fraud_reports = 0

# Open and read the CSV file
with open('2024_CSN_Data_Contributors.csv', mode='r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    
    # Iterate through each row
    for row in reader:
        # Check if the row corresponds to FTC web fraud reports
        if 'FTC - Web Reports (Fraud & Other)' in row['Data Contributor']:
            # Extract the number of reports and add to total
            reports = int(row['# of Reports'].replace(',', ''))
            total_fraud_reports += reports

# Print the total
print(total_fraud_reports)
```

```python
        next(reader)
```
